Proposito del Sitio

El propósito de este sitio es solucionar la problemática de la educación, surgida por la pandemia Covid-19, dando a conocer información sobre las etiquetas semánticas y como estas pueden ayudarnos a definir la estructura de un documento el cual indicaría la organización del sitio web, junto con esto también se tiene como propósito informar sobre la técnica CSS grid y su uso para los desarrolladores web; todo esto con el propósito de que los estudiantes puedan tener esta información sin la necesidad de salir de sus hogares durante la pandemia.

Etiquetas Semanticas

Las etiquetas semánticas son aquellas que nos indican la estructura de un documento, indican a un navegador donde debe ir el contenido y como debe ir, por ejemplo: al usar la etiqueta B indica que el texto debe ir en negrita y si se usa la etiqueta P indica que es un párrafo. Son estas etiquetas las que forman la estructura de un sitio web, indicando si es un título o un párrafo lo que el navegador debe interpretar en cada parte de este; es decir indican que es y cual es el significado de los elementos que estén allí dentro, Existen varias etiquetas semánticas que definen el esquema principal del documento o sitio web, como lo son: HEADER, FOOTER, SECTION, ARTICLE, ASIDE o NAV. Entre estas conforman la cabecera y el pie de página, también su barra de navegación y muchas mas cosas que hacen que el navegador interprete de una forma precisa la información y elementos que le estemos dando. “Etiquetas semánticas del HTML5. (2012, June 4). Desarrolloweb.com; Desarrollo Web. https://desarrolloweb.com/articulos/etiquetas-semanticas-html5.html”

CSS Grid

CSS Grid es una técnica o un sistema de hojas de estilo cascada, que su función principal es permitir que los desarrolladores web tengan la oportunidad de crear diseños complejos los cuales sean adaptables a la mayoría de navegadores. CSS Grid no es actualmente el estándar, pero ha logrado ser adoptada por los principales navegadores. CSS Grid ha sido capas de crear diseños más asimétricos y brindando opciones como CSS Float; junto con esto también ha permitido un código mas estandarizado el cual es el que permite que sea compatible con la mayor parte de navegadores. “de, C. (2020, May 13). CSS grid layout. Wikipedia.org; Wikimedia Foundation, Inc. https://es.wikipedia.org/wiki/CSS_grid_layout”
